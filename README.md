# 🎮 Simon Says Game

A fun, interactive browser-based memory game built with **JavaScript, HTML, and CSS**. The goal is to repeat the sequence of colors shown by the game. Each round adds a new step, testing your memory and focus!

---

## 🚀 Features
- Dynamic color sequence generation  
- Increasing difficulty with each level  
- Simple and responsive design  
- Pure JavaScript implementation (no external libraries)  

---

## 📂 Project Structure
Simon-Says-Game/
│── index.html      # Main HTML file
│── style.css       # Styling for the game
│── app.js          # Game logic in JavaScript
│── README.md       # Project documentation

## 📦 Installation & Setup
Clone the repository and open the game in your browser:

```bash
git clone https://github.com/Pratikshyadash1707/Simon-Says-Game.git
cd Simon-Says-Game

Then simply open index.html in your preferred browser.

▶️ How to Play
1. Press any key to start the game.

2. Watch the sequence of colors   highlighted.

3. Repeat the sequence by clicking the colored buttons.

4. Each round adds a new color to the  sequence.

5. If you make a mistake, the game ends and you can restart.
